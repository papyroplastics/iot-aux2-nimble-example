
# NimBLE example

Este es un ejemplo muy minimalista de como hacer un servidor GATT con NimBLE en ESP-IDF, su proposito es únicamente didactico, no recomiendo usarlo como base para sus tareas, ya que está bastante incompleto y no contiene todos las caracteristicas que van a necesitar utilizar, además no tiene checkeo de erorres en ningúna parte.

Para ejecutar se deben seguir las intrucciónes del pdf del aux 1, para testear recomiendo instalar la applicación **nRF Connect for Mobile** en sus telefonos, la cual permite conectarse al dispositivo e interactuar con los servicios GATT.
